**Map Harvest** is a professional landing page for the Map Harvest Chrome Extension—a powerful, privacy-first tool designed to scrape Google Maps search results directly into cleanly formatted CSV files. No backend, no API keys, and zero data leaves your browser.

> [!NOTE]
> This repository contains the official landing page and legal documentation for the Map Harvest ecosystem.

---

## ✨ Key Features

- **🎯 Automated Scraping**: Effortlessly extract business names, phones, websites, and ratings.
- **🔍 Deep Insights**: Automatically fetches business websites in the background to find emails and social media links.
- **📊 Excel-Ready Data**: Smart formatting ensures phone numbers and special characters are preserved perfectly in Excel/Sheets.
- **🔒 Privacy First**: 100% client-side execution. Your data stays on your machine.
- **🌓 Modern Aesthetics**: Built with the **Sahed Design System**, featuring a premium dark mode, bento-grid layouts, and smooth GSAP animations.
- **📱 Fully Responsive**: Optimized for a flawless experience across desktop, tablet, and mobile devices.

---

## 🚀 Tech Stack

- **Structure**: Semantic HTML5
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) for a sleek, modern UI.
- **Animations**: [GSAP](https://greensock.com/gsap/) for high-performance scroll reveals and micro-interactions.
- **Typography**: Inter & JetBrains Mono for a technical, premium feel.
