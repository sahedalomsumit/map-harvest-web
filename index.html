Map Harvest Extension

A lightweight, fully client-side Chrome Extension that scrapes Google Maps search results and exports the data directly into a cleanly formatted CSV file. No backend, no API keys, and no rate limits. 100%_CLIENT_SIDE

View on GitHub
/ SYSTEM_CAPABILITIES

Key Features

01 Automated Scraping

Automatically scrolls through Google Maps search results, clicks each listing, and extracts the data. Captures business name, phone number, website, total reviews, and review score.

02 Deep Scraping

Fetches the business's website in the background to extract email addresses and social media links (Facebook, Instagram, LinkedIn) while avoiding CORS issues.

📊 Excel-Friendly CSV

Phone numbers are automatically prepended with their country code and formatted so that Excel reads them as text, preventing them from being converted into scientific notation.

🔒 100% Client-Side

All scraping and CSV generation happens directly in your browser. No data is sent to external servers, and no API keys are required.

/ GETTING_STARTED

How to Use

STEP_01

Install Extension

Clone the repository, enable Developer mode in Chrome extensions, and load the unpacked folder.

STEP_02

Search Maps

Go to Google Maps and search for any query (e.g., "dentists in Helsinki", "spas in Dublin").

STEP_03

Start Scraping

Click the extension icon, hit "Start Scraping", and watch the data pour in. Export to CSV when done!
